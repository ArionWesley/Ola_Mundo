Tags em HTML

Na linguagem HTML praticamente tudo é feito atraves de tags. Por isso, é importante entender quais delas estão relacionadas às principais praticas cotidianas. Para tanto, fica aqui uma lista de ações muito importantes e corriqueiras bem como suas tags correspondentes:

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Código-Fonte
  • Listas Numeradas
  • Listas Demarcadas
